# Chat-me
Simple real-time chat with node, express, socket.io

**Requeriments:**<br>
1. Nodejs

**Instructions:**<br>
1. Install dependencies, In your project folder run in your terminal<br>
`npm install`<br>
2. Run the server<br>
`node server.js`<br>
3. Ready! Open your server in your web browser on port 8888.<br>`localhost:8888`
